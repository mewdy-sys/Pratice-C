•	Что такое анонимные типы? 
	Объекты без явного определения типа, полезны дял хранения данных для LINQ запросов
•	Что такое LINQ? 
	Инструмент, который позволяет обрабатывать данные в виде SQL запросов
•	Что такое выражение запроса? 
	Запрос, который используется для обработки данных, использует SQL синтаксис
•	Что такое переменная запроса? 
	Переменная, которая хранит результат запроса, не выполняет запрос до обращения к ней
•	Что такое переменная диапазона? 
	Переменная, которая используется внутри запроса, например для перебора элементов
•	Что такое динамический тип? 
	Тип, который определяется во время выполнения программы, могут хранить значения любого типа
•	Что делает оператор dynamic? 
	Указывает, что переменная динамического типа, позволяет хранить значения любого типа
