•	Что такое обобщение? 
•	(Generics) ?
•	Что такое закрытый тип? 
	(Closed Type) обобщенный тип, в котором все параметры типа заменены на конкретные типы. List<int>
•	Что такое открытый тип? 
	(Open Type) обобщенный тип, в котором один или несколько параметров типа не заменены на конкретные типы. List<T>
•	Объясните понятия ковариантности и контравариантности обобщений. 
    Ковариантность позволяет использовать производный вместо базового типа. Контравариантность позволяет использовать более общий вместо производного типа.
•	Какие преимущества использования обобщений? 
    Универсальные классы и методы, не нужно упаковывать и распаковывать значения, безопасность типов.
•	Что такое Nullable тип? 
	Позволяет значимым типам принимать Null
•	Что такое операция поглощения? 
	позволяет задать значение по умолчанию, если значение Null
•	Какие вы знаете типы ограничений для обобщений? 
	1.	where T : struct: Ограничение на значимые типы.
	2.	where T : class: Ограничение на ссылочные типы.
	3.	where T : new(): Ограничение на типы с публичным конструктором без параметров.
	4.	where T : <base class>: Ограничение на типы, которые наследуются от указанного базового класса.
	5.	where T : <interface>: Ограничение на типы, которые реализуют указанный интерфейс.
public class GenericClass<T> where T : class, new()
{
    public T CreateInstance()
    {
        return new T();
    }
}
